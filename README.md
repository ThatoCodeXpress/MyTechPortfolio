# MyTechPortfolio

**MyTechPortfolio** is a personal portfolio web application showcasing my journey as a full‑stack and AI‑powered developer.  
It highlights real solutions I’ve built — from banking and e‑commerce systems to ride‑hailing, smart alarms, and service apps.  
This portfolio reflects my applied AI, DevOps, and full‑stack skills, with direct access to my CV for recruiters and companies.

---

## 🚀 Features
- Single HTML file — contains all code (HTML, CSS, JavaScript).
- Dark theme UI — modern design with animations, hover effects, and responsive layouts.
- Embedded images — project screenshots and profile photo are base64 encoded, so no external files are needed.
- Interactive navigation — hamburger menu, scroll‑to‑top button, and scroll reveal animations.
- Contact form — simple form with JavaScript validation.
- Offline ready — works without internet or server setup.

## 📄 How to Run
1. Download `index.html`.  
2. Double‑click the file — it opens directly in your browser (Chrome, Edge, Firefox).  
3. No installation, server, or internet required.

## 🌐 Deployment
You can host this portfolio easily:
